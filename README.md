# UnnamedProyect_Backend

## Install dependencies

```
npm run install
```

## Create environment variables

* Create .env file
* Add next variables:
  ```
  SERVER_PORT=3000

  # DATABASE VARIABLES
  DB_HOST=localhost:5432
  DB_NAME=postgresdb
  DB_USER=admin
  DB_PASSWORD=admin
  ```

## Start projects
```
node src/app.js
```
